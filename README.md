# URDF Robot Gallery

A lightweight, public list of robot repositories curated for URDF Studio.

## How to submit

Use the submission form:
https://github.com/urdf-studio/urdf-robot-gallery/issues/new?template=robot-repo-submission.yml

## Data source

The gallery reads from `docs/robots.json`.
